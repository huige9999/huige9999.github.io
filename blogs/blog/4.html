    svg实现打勾动画

    汪元会 7/19/2024 screenHd技术亮点svg

    # 效果演示

    # 代码解析

    # 1. 核心思路

    step1:用 svg 画一个中间有勾的圆的静态效果 step2：通过设置 css stroke-dasharray 和 stroke-dashoffset 属性实现打勾动画效果

    # 2. 难点剖析

    这里的难点在于 stroke-dasharray 和 stroke-dashoffset 属性的使用。 下面是圆形的样式代码

         .circle {
        stroke-dasharray: 1194;
        stroke-dashoffset: 1194;
      }

       @keyframes circle {
        from {
          stroke-dashoffset: 1194;
        }

        to {
          stroke-dashoffset: 2388;
        }
      }

    # 4. 总结

    • stroke-dasharray：描述实线和虚线的长度的数列
    • 数列的数据个数必定是偶数个的，如果是奇数个则复制前面的数构造成偶数 比如(1,2,3 会变成1,2,3,1,2,3)
    • stroke-dashoffset：描述实线的偏移量，实线的起点位置 通过改变偏移量就能实现逐渐出现或隐藏的效果啦！

    <!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

    <style>
      .circle {
        stroke-dasharray: 1194;
        stroke-dashoffset: 1194;
      }

      .circle.animate {
        animation: circle 0.8s ease-in-out forwards;
      }
      @keyframes circle {
        from {
          stroke-dashoffset: 1194;
        }

        to {
          stroke-dashoffset: 2388;
        }
      }

      .tick {
        stroke-dasharray: 350;
        stroke-dashoffset: 350;
      }
      .tick.animate {
        animation: tick 0.6s ease-out 0.7s forwards;
      }

      @keyframes tick {
        from {
          stroke-dashoffset: 350;
        }

        to {
          stroke-dashoffset: 0;
        }
      }
    </style>
  </head>

        fill="none"
        stroke="#34c2e5"
        stroke-width="20"

        cy="200"
        r="190"
        class="circle animate"
        stroke-linecap="round"
        transform="rotate(-90 200 200) "

        stroke-width="24"
        points="88,214 173,284 304,138"
        stroke-linecap="round"
        stroke-linejoin="round"
        class="tick animate"
      />
    </svg>
  </body>
