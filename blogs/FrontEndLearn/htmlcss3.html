    HTML进阶

    汪元会 10/3/2024 html+css

    # iframe元素

    • 在网页中嵌入另一个页面
    • 可替换元素(通常显示的内容取决于元素的属性)

    补充：a元素切换iframe显示的页面

    iframe:name="定义的名字" a:target="定义的名字"

    使用场景：我的网站想要放别人网站的视频

    # 表单元素

    一系列元素，主要用于收集用户数据

    # input元素

    • type属性决定输入框类型 你想要什么类型的数据? 比如text、password

      emmet语法：input:type

    • value属性表示输入框的值 程序要定义或使用的输入框的值

    • placeholder属性表示提示文本

    关于 checkbox多选框、radio单选框 为了从逻辑上对多选框进行分组，我们可以加上name属性

    input元素做按钮 按钮类型包括：重置(reset)、提交(submit)、普通按钮(button)

    # select元素

    select option配合使用，实现下拉列表

    select选项分组 option元素外面套一个optgroup元素

    <select>
<optgroup label="组名">
 <option value="值">选项</option>
</optgroup>
</select>

    # textarea元素

    建议用css样式调整大小

    特殊的地方：textarea虽然也是可替换元素，但是它的内容由写的文本决定而不是value属性

    # 按钮元素

    type: reset(重置)、submit(提交、默认值)、button(普通)

    说明：button是不可替换元素，所以我们可以在button中间写任何内容来丰富按钮的内容(比input做的按钮灵活)

    # 配合表单的其他元素

    • label元素 通常配合单选和多选框使用

    需求：点击文字的时候也能选中单选 理解：label是标签的意思，比如衣服上的标签关联这个衣服，同样的道理，label元素将文字和单选框关联起来

    使用方式：

    1. input定义个id属性值,label的for属性使用它
    2. label套着input和文字
    • datalist元素

    配合input元素使用，实现输入提示功能

    使用方式: datalist配合option定义好若干suggestion选项并定义一个id属性值,input的list属性使用它(两者产生关联)

    • form元素 包含所有表单元素 作用：提交表单的时候可以将表单数据以合适的方式提交到服务器(即：它内部封装了表单提交的必要功能)

    使用：

    1. form的action="提交的地址"、method="提交方式"

    2. input需要加上name属性，作为提交的字段名

    # fieldset元素

    对表单项进行分组,配合legend元素定义组名

    使用：直接用它包裹需要分在一组的表单项即可

    # 表单元素美化

    • outline设置input外边框 outline-offset设置边框偏移量

    • :focus伪类设置输入框聚焦状态下的样式

    • :checked伪类设置单选框和复选框选中状态下的样式 补充：单选/多选框由于是可替换元素，所以一些css样式无法生效(比如background),但我们可以通过它的选中状态来设置它所关联元素的样式!

    • 重置表单元素通常设置border:none、outline:none 然后再设置自己的border样式

    • textarea设置resize属性可以让用户自由调整大小

    • 输入框文本内容离边框距离调整： padding、text-indent

    • 通过css实现一个可点击的单选框思路:

      1. 隐藏真实的单选框
      2. 定义普通元素并给它设置单选框样式
      3. 通过input:checked+.radio设置自定义单选框选中样式
        <label class="radio-item">
          <input name="gender" type="radio">
          <span class="radio"></span>
          <span>男</span>
 </label>


 // 普通元素做成单选框
  .radio-item .radio {
          width: 12px;
          height: 12px;
          border: 1px solid #999;
          border-radius: 50%;
          cursor: pointer;
          display: inline-block;
  }
  // 设置选中状态的样式
    .radio-item input:checked+.radio{
          border-color: #008c8c;
      }
  // 隐藏真实的单选框
   .radio-item input[type="radio"]{
          display: none;
      }
