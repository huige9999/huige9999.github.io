    单调有界收敛准则、夹逼定理、连续与间断、闭区间上连续函数的性质

    汪元会 10/10/2024 高等数学

    # 单调有界收敛准则

    数列Xn单调、有界(增上界、减下界)则极限存在

    # 解题思路

    1. 先看单调性
    2. 在确定对应方向是否有界

    一般在求单调性的过程中，有界性会顺带明确。因为求数列单调性会确定某个不等式的关系，该关系能反映出是否有界

    # 解题小结

    若题目给出递推公式(后一项和前一项的关系)，一般考虑单调有界收敛准则

    对{Xn}放缩的方法：

    • 观察法，用数学归纳法
    • 常见不等式

    x平方+y平方 >= 2xy 当x属于(0,PI/2)时,sinx < x < tanx

    对于放缩的理解：当前表达式A不好求解，表达式B简单，找出A与B的不等式关系，然后用B代替A，再求解B

    # 夹逼定理

    数列：n向右无限延展的过程中，有Xn<=Yn<=Zn，且Xn的极限=Zn的极限=a，那么Yn的极限=a

    函数：x趋向于x0的过程中，有X(x)<=Y(x)<=Z(x)，且X(x)的极限=Z(x)的极限=A,那么Y(x)的极限=A

    夹逼定理的本质是对求极限的数列或函数进行放缩

    # 放缩的原则

    • 不改变极限值
    • 尽量简化计算

    解题思路：估极限 然后观察法

    # 无穷数列的和式极限

    1. 夹逼定理
    2. 定积分定义 标志：求和k从1到n,f(k/n).(1/n)

    • 分子不变的情况： 看首尾

    • 分子变化的情况 step1: 固定分子为k，对分母进行放缩 step2: 三边同时求和

    # 连续与间断

    # 连续的定义

    lim(x->a)f(x) = f(a) 即：函数在某点处的极限值等于函数值则称函数在该点处连续

    连续的条件：1. a处有定义 2. a处极限存在 3.极限值=函数值

    连续的等价定义(增量形式)

    lim(增量x->0)增量y = 0 或lim(增量x->0)f(a+增量x) - f(a) = 0

    # 左右连续

    该点处，左极限或者右极限=函数值

    连续的充要条件：左连续=右连续=函数值

    # 区间连续

    (a,b)区间每一点都连续，则f(x)在(a,b)内连续 a处右连续，b处左连续，则f(x)在[a,b]上连续

    # 连续函数的结论

    • 连续函数的和、差、积、商仍连续
    • 连续函数有限复合仍连续
    • 连续且单调的函数的反函数仍然单调连续
    • 基本初等函数在其定义域连续
    • 初等函数在其定义区间连续

    # 间断点

    f(x)在x0处不连续，则间断(即该点函数值不等于极限值)

    # 间断点的分类

    # 第一类间断点

    定义：左右极限均存在

    继续划分：

    • 可去间断点 左右极限相等 可去间断点可以通过补充定义使其连续 举例：sinx/x的在0处极限为0，函数值不存在，可以补充定义0处函数值为0使其连续 函数图像描述：看上去是连续的，但是某个点是空的

    • 跳跃间断点 左右极限不相等 举例：f(x)= { x^2 + 1(x>=0),-1 (x < 0)} 函数图像描述：某个点处跳跃了

    # 第二类间断点

    定义：左右极限至少一个不存在

    继续划分：

    • 无穷间断点 左右极限至少有一个无穷 函数图像描述：向无穷远处发散

    • 震荡间断点 左右极限均不为无穷 函数图像描述：来回震荡 举例：sin(1/x)在x=0处的极限时而为1，时而为-1

    # 间断点的解题思路

    • 找出函数可去间断点的个数 先找函数无定义的点，然后再从这些点中筛选出极限存在的点(满足未定式)

    # 闭区间上连续函数的性质

    # 1. 最值定理

    f("可c") = m <= f(x) <= M = f("爱它"),"可c","爱它"属于[a,b]

    推论:f(x) 在 [a,b]有界

    # 2. 介值定理

    f(x)可取介于m,M之间的一切值(通常与最值定理联用)

    任意满足 m<=A<=M,存在"可c"属于[a,b]使得f("可c")=a

    注："最+介" 当所证明结论中，中值"可C"属于[a,b]时

    # 3. 零点定理

    端点处函数值异号，则存在"可c"属于(a,b)，使得f('可c') = 0

    # 解题步骤

    1. "可c" -> x
    2. 右边 = 0
    3. 左边 = f(x)
