    函数的定义及运算、性质、类型

    • 定义域、对应法则、值域

    • 定义域和对应法则确定一个函数

    # 函数的运算

    # 四则运算

    函数加减乘除

    # 复合函数

    • 内层函数的值域作为外层函数的定义域 注意：内层的值域要被包含于外层的定义域内

    • 做题的思路 1、内层函数表达式 代入 外层函数的x 2、 如果有多层嵌套，则从内到外一层一层代入计算

    # 反函数

    1、定义：反解出x 它与原函数图像一致，只是反解出x

    2、存在反函数的充要条件 x、y一一对应(即单调)

    3、反函数的性质

    • 原函数和反函数图像关于y=x对称
    • 和单调性一致

    补充：规定反正弦在上有反函数，如果题目给出该区间之外的区间，要你求它的反函数的解题思路： 1、判断该区间函数是否单调 2、如果单调，使用变量代换平移到区间

    # 函数的性质

    # 单调性

    1. 定义 单调递增 单调不减

    2. 性质

    • f(x1),f(x2)增则f(x1)+f(x2)增(加)
    • f(x)增,Cf(x) C>0增 C<0减(乘法)
    • 同增异减(复合)

    3. 判定方法

    数列(定义法)： 后一项减前一项，大于0增，小于0减 后一项除以前一项，大于1增，小于1减

    函数(求导法)： 求导，导数大于0增，小于0减 注：前提是函数在闭区间连续，开区间可导(为啥是开区间可导?因为导数的本质是极限，极限不一定需要取到临界点值)

    补充： 求导的证明：拉格朗日中值定理

    证明：因为,所以，所以单调递增

    # 周期性

    1. 定义 f(x+T)=f(x) nT是所有周期,其中最小正数为：最小正周期

    2. 性质

    • Cf(x)最小正周期不变,f(Cx)最小正周期为T/|C|
    • f1(x),f2(x)周期为T,则k1f1(x)+k2f2(x)周期为T,但最小正周期可能会变 (线性组合周期不变) (加减操作导致最小正周期变) (以三角函数为例：cos2x+sinx最小正周期为pi)

    补充：迪利克雷函数

    f(x) = {1, x为有理,0, x为无理} 它没有最小正周期，周期为任意正有理数

    证明思路：假设T为有理，再讨论x有理、无理情况下的函数值 有理+有理=有理=1，有理+无理=无理=0

    3. 考点1：判定函数是否为周期函数

    结论：抽象的周期函数，T与条件中给出的参数有关系，是其2倍

    证明思路：f(x+T) = f(x)、形似找框

    4. 考点2：原函数求导，原函数和导函数周期性相同，反过来不一定成立

    即：导函数的周期确定，原函数的周期不一定和它相同(举例：导函数cosx+1，原函数sinx+x不是周期函数)

    # 奇偶性

    1. 定义

    前提：定义域为对称区间 f(-x) = f(x) 偶函数 f(-x) = -f(x) 奇函数

    2.常见奇偶函数

    奇函数：(k为奇数),sinx,tanx,cotx,

    偶函数：(k为偶数),cosx,

    补充：

    记忆方法: ln下,普通x+升次再降次的x，当然x旁边再加个常数的平方(不然升次降次的操作无意义)

    该函数求导是:，根号外面的x没了

    3. 性质

    • 偶关于y轴对称,奇关于原点对称,奇函数过原点(图像)
    • ,奇,则奇(线性组合奇偶性不变)
    • ,同偶异奇(相乘)
    • ,,(偶函数) f(x)-f(-x)除以2(奇函数)
    • 任意函数 = (偶函数) + (奇函数)

    总结归纳： 奇函数的负号可以从里面拿到外面、 偶函数的负号可以直接去掉

    4. 考点

    原函数求导，每求一次导就改变一次奇偶性 反之， 若导函数是奇函数，则原函数一定是偶函数(说明：反之一般都不成立，这个是特殊情况) 若导函数是偶函数，原函数不一定是奇函数，除非C=0 (记忆技巧：还原回去是偶函数而偶函数的负号可以直接消掉即：不受外界影响，所以一定成立)

    # 有界性

    1. 定义 m<=f(x)<=M

    2.有界性解题思路总结

    • 遇到式中带1的平方和与平次数，观察它们之间的关系，分析能不能使用基本不等式 比如：和x,可以将x转换成的关系式(通过不等式：)

    • |式子|<常数能立刻确定有界性

    • 遇到三角函数，可以将x用k*pi(即：k的表达式)作变量代换，再去分类讨论

    # 常见函数类型

    1. 基本初等函数：反、对、幂、指、三

    2. 初等函数：基本初等函数组成、一个式子就能表达的函数

    3. 分段函数

    4. 隐函数 变量都写到左边，右边=0的方程形式

    5. 参数方程 x,y用第三方变量(参数t)来表示

    6. 极限函数 求极限得出函数的真实表达式

    7. 积分上限函数 F(x)=积分(下限:a,上限:x)f(t)dt 有两个变量：x,t x在[a,b] t在[a,x]

    结论： f(x)在[a,b]连续,则F(x)在[a,b]上可导，导数为x替换f(t)中的t (记忆技巧: 积分是原函数，原函数求导又回到原函数)
