    两个重要极限、洛必达法则、泰勒公式、未定式的解法总结

    汪元会 10/1/2024 高等数学

    # 两个重要极限

    1. lim x->0 sinx/x = 1
    2. lim x->0 (1+x)^(1/x) = e 记忆：1的无穷次方型、幂指函数 得到e

    # 一个重要不等式

    0<x<π/2时,sinx<x<tanx

    对不等式除以x、再求极限可以推出第一个重要极限

    小结归纳：比较三角函数的大小，可以通过画单位圆然后标注出对应的三角函数来比较。

    # 1的无穷次方型的幂指函数求极限

    1. 重要极限2
    2. 幂指函数变形(形如:f(x)^g(x))

    e^lnf(x)^g(x) = e^g(x)lnf(x) 通过取e为底的对象将右上角的函数移到左边

    # 洛必达法则

    0/0或无穷/无穷型可以用洛必达法则来比较。

    方法：对分子、分母同时求导再求极限

    注意点：求导后的极限存在或者为无穷(不存在的一种情况，例如：lim x->0 sin(1/x))

    # 无穷大间的比较

    x->无穷,linx << x << e^x

    补充：sinx+cosx为有界变量

    # 泰勒公式

    f(x) = f(x0) + f(x0)的1阶导/1!(x-x0) + f(x0)的2阶导/2!(x-x0)^2 + ... + f(x0)的n阶导/n!*(x-x0)^n + Rn(x0)

    意义：将函数转换成幂函数的形式， 方便求导、积分。 记忆技巧：f(x)在x0点处进行展开，求导/阶层*距离的平方

    1. 皮亚诺余项

    Rn(x) = O[(x-x0)^n]

    用于求极限

    1. 拉格朗日余项

    Rn(x) = f(a)的n+1阶导数 / (n+1)! * (x - x0)^(n+1),a介于x0和x之间

    用于证明题

    当x0=0时，称为麦克劳林展开式

    x->0时的常用泰勒公式公式：

    • e^x = 1 + x + 1/2!*x^2 + 1/3!*x^3 + ... + 1/n!*x^n + O[x^n] 直接代入泰勒公式

    • sinx = x - 1/3!*x^3 + 1/5!x^5 - .... + (-1)^n(1 / (2n+1)) * x^(2n+1) + O[x^(2n+1)] 直接代入泰勒公式

    • cosx = 1 - 1/2!*x^2 + 1/4!x^4 - .... + (-1)^n(1 / (2n)) * x^(2n) + O[x^(2n)] sinx求导

    • ln(1+x) = x - 1/2x^2 + 1/3x^3 - .... + (-1)^(n+1)*(1 / n) * x^n + O[x^n] 间接法推导(联想到导数)，通过求1/1+x=多项式的和(通过等比数列和公式)

    • arctanx = x - 1/3x^3 + 1/5x^5 - .... + (-1)^(n)*(1 / (2n+1)) * x^(2n+1) + O[x^(2n+1)] 也是间接法推导(联想到导数)，x^2代入上式

    • (1+x)^a = 1 + ax + a(a-1)/2! * x^2 + ...+a(a-1)...(a-(n-1)) / n! * x^n + O[x^n] 二项式展开定理
